== Installation ==

= Minimum Requirements =

* WordPress at least 3.0.1 or greater
* PHP version 5.2 or greater

https://www.youtube.com/watch?v=qDql4GJXYWI

= Installation =

1. Login to your [ProductDyno](https://app.productdyno.com/ "ProductDyno") account, click on left navigation link 'Wordpress Plugin' and download it. Upload `productdyno.zip` which you have just downloaded to the `/wp-content/plugins/` directory. 
2. You may also search ProductDyno plugin on wordpress plugins search durectory by keyword `productdyno`.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Find ProductDyno on left navigation.
5. Login to your [ProductDyno](https://app.productdyno.com/ "ProductDyno") application and Click on My Account on left and copy your API Key
6. Back to your ProductDyno Plugin section, Click the connect box with unlock icon.
7. Enter your API key.
8. Create a new page or Edit any existing page, you will see ProductDyno box in top right corner.
9. Select any product/collection which you have created on [ProductDyno](https://app.productdyno.com/ "ProductDyno") from dropdown
10. Hurray! That's it. 
 
== Frequently Asked Questions ==
 
= How to get the API key? =
 
Login to your [ProductDyno](https://app.productdyno.com/ "ProductDyno") account, click "My Account" from left navigation and copy your API key from this page.
 
= How many membership sites can I secure with this plugin on a single domain? =
 
As many as you want. Unlike other membership plugins, we do not restrict you to running only one secure membership on a single domain.

= Do I need an active ProductDyno account to use the plugin? =
 
Yes, you will need an active [ProductDyno](https://productdyno.com/ "ProductDyno") account.

= My question is not answered here, what should I do? =
 
Please contact our support and we will be happy to help you out.
